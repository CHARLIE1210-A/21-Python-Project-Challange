# Password Generator

This is a simple and secure password generator written in Python as part of the "21 Days Python Project Challenge". It allows you to generate strong, random passwords with customizable options and provides a strength rating for each password.

## Features
- Generate one or multiple passwords at a time
- Choose password length (between 4 and 64 characters)
- Select which character types to include:
  - Uppercase letters
  - Lowercase letters
  - Digits
  - Symbols
- Password strength evaluation (Very Weak to Excellent)

## How to Use
1. Run the script:
   ```bash
   python main.py
   ```
2. Enter how many passwords you want to generate.
3. Enter your preferences for password length and character types.
4. The generated passwords and their strength ratings will be displayed.

## Example
```
=== Password Generator ===
How many passwords to generate? 2
Password length: 12
Include upercase letters? (y/n): y
Include lowercase letters? (y/n): y
Include digits letters? (y/n): y
Include symbols letters? (y/n): y
Password 1: 8@kL2!dPzQw# | Strength: Excellent
Password 2: 7gH$1mN!qR2& | Strength: Excellent
```
