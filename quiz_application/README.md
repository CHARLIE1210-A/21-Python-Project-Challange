# Python Quiz Application

This is Project 7 of the **21 Python Projects Challenge**.

## Description
A simple command-line Python quiz application that tests your knowledge of Python basics. The quiz presents multiple-choice questions, checks your answers, calculates your score, and allows you to retake the quiz as many times as you like.

## Features
- Multiple-choice questions about Python
- Immediate feedback on each answer
- Score calculation and percentage display
- Option to retake the quiz

## How to Run
1. Make sure you have Python installed (version 3.x).
2. Download or clone this repository.
3. Open a terminal and navigate to the project directory.
4. Run the following command:
   
   ```bash
   python main.py
   ```
