# Special Tetris – 21 Python Challenge Project

This project is a simplified version of the classic Tetris game, created as part of the "21 Project Python Challenge." It uses the Pygame library to provide a graphical interface and interactive gameplay.

## Features
- Classic Tetris gameplay with 5 different tetromino shapes (I, O, T, L, J)
- Colorful blocks and grid
- Score tracking based on lines cleared
- Keyboard controls for moving and rotating pieces

## Controls
- **Left Arrow**: Move piece left
- **Right Arrow**: Move piece right
- **Down Arrow**: Move piece down faster
- **Up Arrow**: Rotate piece
- **Close Window**: Quit game

## Requirements
- Python 3.x
- Pygame

## Installation
1. Install Python 3 if not already installed.
2. Install Pygame:
   ```bash
   pip install pygame
   ```

## How to Run
1. Download or clone this repository.
2. Open a terminal in the project directory.
3. Run the game:
   ```bash
   python main.py
   ```

## Project Structure
- `main.py` – Main game file containing all logic and rendering

This project is part of the "21 Project Python Challenge," designed to help learners build practical Python skills through hands-on projects.
